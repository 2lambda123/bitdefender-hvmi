The purpose of this document is to provide the basic steps that need to be taken in order to generate Doxygen documentations.
	
	For Windows:
	1)	Install Doxygen http://www.doxygen.nl/download.html
	2)	Download Graphviz (*.zip) and add the Grapwiz/bin folder into the Path environment variable https://www.graphviz.org/download/
	3)	Run make_doxy.cmd
	4)	Open Doxygen/html/index.html
